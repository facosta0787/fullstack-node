APIRest User administration

API Reference

Unprotected Routes

POST /api/signup -> Use to create user. to create user as admin you should be an Admin User

{ user: "test", name: "test name", password: "testpassword", admin: true or false }

POST /api/signin -> Use to login into the system ang get your JSon-Web-Toker

{ user: "test", password: "testpassword" }

Protected Routes

GET /api/post -> Use to fetch all posts. You have pass Authorization Header with the JWToken

Authorization Bearer youShouldPutYourJsonWebTokenForWebAuthentication

GET /api/post/{ id } -> Use to fetch on post. You have pass Authorization Header with the JWToken

Authorization Bearer youShouldPutYourJsonWebTokenForWebAuthentication

POST /api/post -> Use to create one post. You have pass Authorization Header with the JWToken

Authorization Bearer youShouldPutYourJsonWebTokenForWebAuthentication { title: "lorem ipsum title", text: "lorem ipsun text or description" }

DELETE /api/post -> Use to delete one post. You have pass Authorization Header with the JWToken

Authorization Bearer youShouldPutYourJsonWebTokenForWebAuthentication { id: 1 }
